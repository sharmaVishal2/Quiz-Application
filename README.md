# Quiz Application

A **Spring Boot** based Quiz Application that allows users to take quizzes, view questions, and submit answers. Built using **Spring Boot**, **JPA/Hibernate**, and **MySQL**, this project demonstrates full CRUD operations and RESTful API design.

---

## **Features**

- Create quizzes with a specific category and number of questions.
- Fetch quiz questions by quiz ID.
- Submit quiz responses and get the score immediately.
- Randomized selection of questions based on category.
- Support for multiple categories like Java, Python, etc.
- Fully RESTful APIs for integration with front-end applications (React, Angular, or plain HTML).

---

## **Technologies Used**

- **Backend:** Spring Boot, Spring Data JPA, Spring Web
- **Database:** MySQL
- **Build Tool:** Maven
- **Other Libraries:** Lombok
